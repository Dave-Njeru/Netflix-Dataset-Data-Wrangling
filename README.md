# Netflix-Dataset-Data-Wrangling
📌 **Project Overview**

This project focuses on cleaning and transforming the Netflix dataset obtained from Kaggle. The goal is to apply data wrangling techniques to prepare the dataset for analysis and visualization. The dataset contains information about movies and TV shows available on Netflix.

Dataset source: <br>
👉 [Netflix Shows Dataset](https://www.kaggle.com/datasets/shivamb/netflix-shows)

---
🎯 **Objectives**

The main objectives of this project are:
- Load the Netflix dataset from a CSV file using Pandas
- Explore and understand the dataset structure
- Perform data discovery (data types, missing values, quality issues)
- Clean the dataset (Handle duplicates, Address missing values)
- Transform and enrich the dataset
- Validate the cleaned dataset
- Export the final cleaned dataset to a `.csv` file

---
🛠️ **Tools and Libraries Used**
- Python
- Kaggle
- Pandas

---
📂 **Project Workflow**
1. **Data Loading**
   - Import the dataset using pandas
2. **Data Exploration**
   - Inspect dataset structure
3. **Data Cleaning**
   - Handling missing values
   - Removing duplicates
   - Fixing formatting issues
4. **Data Transformation & Enrichment**
   - Feature engineering
   - Filtering & sorting
   - Grouping and Aggregation
5. **Data Validation**
   - Check for remaining null values
   - Verify correct data types
   - Ensure no duplicate records
6. **Export the dataset for further use**

---
📊 **Output**
- Final cleaned dataset in CSV format, ready for data analysis or visualization

---
▶️ **How to Run the Project**
1. Open the notebook in Kaggle
2. Download the dataset from Kaggle
3. Install dependencies if needed:
   ```bash
   pip install pandas
   ```
4. Run all cells step by step
5. Export the cleaned dataset
